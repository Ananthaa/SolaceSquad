# Soul Squad - Python Backend (FastAPI + Jinja2)

A full-stack wellbeing platform built with **Python** at its core, using FastAPI for the backend and Jinja2 for server-side rendered templates.

## ğŸš€ Getting Started

### Prerequisites

- Python 3.11 or higher
- pip (Python package manager)

### Installation

```bash
# Navigate to backend directory
cd backend

# Create virtual environment (recommended)
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py

# Or use uvicorn directly
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Access the Application

- **Home Page**: http://localhost:8000
- **User Dashboard**: http://localhost:8000/app
- **Consultant Dashboard**: http://localhost:8000/consultant
- **Login**: http://localhost:8000/login
- **Signup**: http://localhost:8000/signup
- **API Docs**: http://localhost:8000/docs (auto-generated by FastAPI)

## ğŸ› ï¸ Development

### Adding New Routes

Edit `main.py`:

```python
@app.get("/new-page", response_class=HTMLResponse)
async def new_page(request: Request):
    return templates.TemplateResponse(
        "pages/new_page.html",
        {"request": request, "page_title": "New Page"}
    )
```

### Creating Templates

1. Create file in `templates/pages/`
2. Extend base or layout:
   ```jinja2
   {% extends "base.html" %}
   {% block content %}
   <!-- Your content -->
   {% endblock %}
   ```

### Adding Static Files

- CSS: `static/css/`
- JS: `static/js/`
- Images: `static/images/`

Reference in templates:
```jinja2
<link rel="stylesheet" href="{{ url_for('static', path='/css/custom.css') }}">
```
